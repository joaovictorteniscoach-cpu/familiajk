Nuvem — os dois sempre com a versão mais nova

Liga o app (contas + investimentos) a um banco de dados grátis. Os dois usam o mesmo link e veem sempre o último estado.

sincronização desligada

Como criar a nuvem grátis (uma vez só, ~5 min)

  • Acesse console.firebase.google.com e clique em Adicionar projeto (pode pular o Analytics).
  • Abra Build → Realtime Database → Criar banco de dados, escolha a localização e o modo de teste.
  • Copie a URL e cole aqui acrescentando /jk/um-codigo-secreto.json no fim — ex.: .../jk/familia-jk-7x9.json.
  • Marque Sincronizar na nuvem e clique Enviar agora.

Quem tiver a URL completa consegue ver/editar — escolha um código secreto difícil e não publique. Mesmo com a nuvem, faça Backup de vez em quando.

Como usar (e não perder nada)

Feito para vocês dois.
  • Dois modos no topo: 🏠 Contas (dourado) e 📈 Investimentos (verde-água). Tudo separado, num app só.
  • Salva sozinho neste navegador. Toque em qualquer número que ele já vem selecionado — é só digitar por cima.
  • Para nunca perder: use Backup às vezes (baixa um .json) e Importar para restaurar. Backups antigos do Cofre ou do app de Contas são reconhecidos automaticamente.
  • Compartilhar: gera um link com tudo embutido (contas + investimentos). Quem abrir carrega no próprio aparelho.
  • Conselhos (no modo Contas) analisa contas e investimentos juntos e dá recomendações a cada abertura.
  • Nuvem mantém os dois celulares iguais.
